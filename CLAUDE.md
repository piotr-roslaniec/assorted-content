# Agent Instructions

This repo collects weekly one-pagers of assorted content on software engineering and related topics.

## Your Role

- Help write a weekly one-pager summarizing the week's readings
- Write brief, accurate descriptions of each piece of content
- Organize entries by topic within each one-pager
- Use `TEMPLATE.md` as the base for each new week

## Guidelines

- Descriptions should be 1-3 sentences — capture the key insight or takeaway
- Group readings under topic headings (e.g., Systems, Security, PLT, Distributed Systems, Career)
- Keep tone neutral and informative; avoid hype
- Aim for a final one-pager that takes no more than 1 hour to read end-to-end — be selective, favor quality over quantity
- Include source links where available
- If a reading spans multiple topics, place it under the most relevant one
