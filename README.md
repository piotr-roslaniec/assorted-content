# Assorted Content

Weekly assorted content on software engineering and related topics.

Each week's one-pager summarizes articles, papers, blog posts, videos, and other materials worth consuming, organized by topic with brief, accurate descriptions. The goal is a curated selection that takes no more than 1 hour to read through.

## Structure

- `TEMPLATE.md` — Template for weekly one-pagers
- `2026/` — Weekly readings by year
